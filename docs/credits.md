# 平台支持

本平台由**[中国科学技术大学计算机科学与技术学院](https://cs.ustc.edu.cn/)**主办。另外，以下团体提供了非常重要的支持：

- **[计算机实验教学中心](https://cslab.ustc.edu.cn/)**承办本平台，提供服务器、机房、校园网接入以及 `vlab.ustc.edu.cn` 域名
- **[校 Linux 用户协会](https://lug.ustc.edu.cn/)**提供技术支持与指导
